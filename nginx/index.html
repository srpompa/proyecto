Google Chrome

Google Chrome es un navegador web desarrollado por Google, lanzado en 2008. Hoy en día, Chrome es el navegador web más popular del mundo.

Mozilla Firefox

Mozilla Firefox es un navegador web de código abierto desarrollado por Mozilla. Firefox es el segundo navegador web más popular desde enero de 2018.
